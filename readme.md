# Babel Babylon Webpack Flow Boilerplate
Boilerplate for Babel + Babylon + Flow + Webpack Combo.

## What's in here?
- Babel 7
- Webpack 4
- Babylon 3.3.0
- Flow Integeration
- CSSModules & SASS

## Getting Started
Note that your browser must be support WebGL.

Install Dependency.

```bash
$ npm install
```

Run NPM Script.

```bash
$ npm run dev
```

## Commands
Development build

```bash
$ npm run build
```

Dev build + Analyzing

```bash
$ npm run build-analyze
```

Dev build + Dev Server

```bash
$ npm run dev
```

Production build

``` bash
$ npm run build-prod
```
